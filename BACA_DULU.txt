## 💡 LOGIKA PEMBAYARAN (PENTING!)

Dashboard mengecek pembayaran dengan logika ini:

```
SETIAP KALI USER BUKA DASHBOARD:

1. Ambil bulan sekarang: format MM-YYYY
   Contoh: 10-2025 (Oktober 2025)

2. Cari di database:
   Payment.where('month_period', '10-2025')
          .where('status', 'approved')
          .where('student_id', user.student_id)

3. Jika ADA:
   ✅ Tampilkan alert HIJAU "Pembayaran Sudah Lunas"

4. Jika TIDAK ADA:
   ❌ Tampilkan alert MERAH "Pembayaran Belum Lunas"
```

## 🎨 CUSTOMIZATION MUDAH

### Mengubah Teks "Tentang Kami":
Buka: `resources/views/dashboard/student.blade.php` (line ~280)
```html
<p class="text-gray-600 leading-relaxed mb-4">
  <!-- GANTI TEKS INI -->
  Alwi College didirikan pada tahun 2023...
</p>
```

### Menambah Gambar Sekolah:
1. Copy gambar ke: `public/images/sekolah.jpg`
2. Edit view (ganti div placeholder dengan img tag)
```html
<img src="{{ asset('images/sekolah.jpg') }}" 
     alt="Sekolah" 
     class="w-full h-full object-cover rounded-2xl">
```

### Mengubah Warna Card:
Ubah class Tailwind:
```html
<!-- Dari -->
<div class="bg-blue-100">
  <svg class="text-blue-600">

<!-- Menjadi -->
<div class="bg-red-100">
  <svg class="text-red-600">
```

## 📊 DATABASE REQUIREMENTS

Pastikan tabel `payments` memiliki:

```sql
- id (int)
- student_id (int) ← FK ke students
- month_period (varchar) ← FORMAT: MM-YYYY (contoh: 10-2025)
- amount (decimal)
- proof_path (varchar)
- status (enum: 'approved', 'pending', 'rejected')
- note (text, nullable)
- created_at (timestamp)
- updated_at (timestamp)
```

**PENTING:** Column `month_period` HARUS format `MM-YYYY`
